README for the FedICT eID Applet SDK
====================================

=== 1. Introduction

This package contains the FedICT eID Applet SDK. It includes the following
primary artifacts:

- eid-applet-dev-guide.pdf	The eID Applet developer's guide.
- eid-applet-package-x.x.x.jar	The FedICT signed eID Applet JAR itself.
- eid-applet-service-x.x.x.jar	The eID Applet Service JAR.
- eid-applet-service-spi-x.x.x.jar
				The eID Applet Service SPI JAR.
- eid-applet-shared-x.x.x.jar	Required JAR for the eID Applet Service.
- lib/				Contains the 3rd party JAR artifacts.
- eid-applet-service-javadoc/	The javadoc of the eID Applet Service.
- eid-applet-service-spi-javadoc/
				The javadoc of the eID Applet Service SPI.

We suggest to start by reading the eID Applet developer's guide.
